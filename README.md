An E-Commerce test automation framework built with Playwright and Cucumber.

## Dependencies

Ensure you have the following software installed on your machine:
* **Node.js** (v16 or higher)
* **Git**
* **VS Code** (Recommended IDE)

## Project Overview

* **BDD Approach:** Uses **Cucumber** (Gherkin) for human-readable feature files and collaborative testing.
* **Page Object Model (POM):** Modular and maintainable code structure (`src/pages`).
* **Data-Driven Testing:** Supports Scenario Outlines for testing multiple data sets (e.g., Checkout flows).
* **Smart Waits:** Implements Playwright’s auto-waiting mechanism for stability.
* **Reporting:** Generates detailed HTML test reports.

## Tech Stack

* **Core:** TypeScript
* **Browser Automation:** Playwright
* **Test Runner:** CucumberJS
